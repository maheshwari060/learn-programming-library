# 🌿 Git & GitHub Learning Resources

## 📖 Introduction

Git is a distributed version control system used to track changes in source code during software development. GitHub is a cloud-based platform that hosts Git repositories and enables collaboration among developers.

# ⭐ My Recommendation

If you are a beginner, follow this order:

1. Learn Git Basics
2. Create your first repository
3. Commit changes
4. Push to GitHub
5. Learn Branching
6. Learn Pull Requests
